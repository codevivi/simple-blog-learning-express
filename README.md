### 🏁 Getting started

**Must have [Node.js](https://nodejs.org)** installed

1. Clone the repo
2. Go into project directory and Install NPM packages

   ```sh
   npm install
   ```

3. run index.js for help

   ```sh
    node .
   ```

   or

   ```sh
    npm start
   ```
